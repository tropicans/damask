## Verification Plan

### Automated Tests
- Create tests in `backend/app/tests/test_role_auth.py` to:
  1. Register a user with role `"user"` and verify that accessing `/api/jobs/` or `/api/jobs/stats` returns `403 Forbidden`.

- Run tests:
