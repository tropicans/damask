<threat_model>
### Security Threat Model (ASVS L1)
- **Threat**: Uploading unexpected malicious file formats to the revert endpoint.
  - **Mitigation**: Perform frontend extension checks rejecting files that are not `.csv`/`.xlsx` (for data) or `.json` (for key) before initiating requests.
</threat_model>
