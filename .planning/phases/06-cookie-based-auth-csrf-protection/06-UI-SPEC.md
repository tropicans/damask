## Visual Hierarchy & Focal Points

- **Focal Point (Primary Screen):** The file Dropzone card acts as the primary visual anchor when no file is uploaded.
- **Focal Point (Preview Screen):** The interactive Preview Table is the primary visual anchor after file upload.
- **Focal Point (Auth Screen):** The AuthForm card centered on a dark canvas.
- **Accessibility:** All icon-only actions (such as file deletion or download buttons) feature text labels or descriptive tooltip fallbacks.
