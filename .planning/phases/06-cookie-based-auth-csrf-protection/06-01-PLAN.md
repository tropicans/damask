<task type="auto">
  <name>Update backend registration, login, and logout endpoints</name>
  <files>backend/app/api/endpoints/auth.py</files>
  <action>
    Update `login` endpoint:
      - Change the return model to return `UserResponse` instead of `Token`.
      - Generate JWT access token.
      - Call `response.set_cookie(key="secure_data_session", value=access_token, httponly=True, secure=settings.COOKIE_SECURE, samesite="lax", max_age=86400)` where `response` is an instance of FastAPI `Response` added to the route parameters.
      - Return the authenticated `user` object directly.
    Update `register_user` endpoint:
      - Change endpoint to accept `response: Response` as a parameter.
      - After creating user and hashing password, generate JWT access token.
      - Set `secure_data_session` cookie on response.
      - Return `user` object directly.
    Add a new `POST /logout` endpoint:
      - Accepts `response: Response` and `current_user: User = Depends(get_current_user)`.
      - Calls `response.delete_cookie(key="secure_data_session", httponly=True, secure=settings.COOKIE_SECURE, samesite="lax")` (sets Max-Age=0).
      - Returns `{"message": "Keluar berhasil"}`.
  </action>
  <verify>
    Run backend and verify auth endpoint routing behaves correctly.
  </verify>
  <acceptance_criteria>
    - login sets secure_data_session cookie and returns UserResponse.
    - register_user sets secure_data_session cookie and returns UserResponse.
    - POST /logout exists and deletes secure_data_session cookie.
  </acceptance_criteria>
  <done>Authentication endpoints set, modify, and delete the HttpOnly session cookie.</done>
</task>
