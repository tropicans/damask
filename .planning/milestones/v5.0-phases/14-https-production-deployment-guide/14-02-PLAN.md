    2. VPS Setup & Hardening:
       - Install Docker & Docker Compose.
       - Configure UFW firewall (default deny incoming, default allow outgoing, allow only port 22/SSH).
    3. Production Environment Variables Checklist:
       - JWT_SECRET_KEY (production-grade random hex).
       - POSTGRES_PASSWORD (secure string).
       - DATABASE_URL (internal network URL).
       - CORS_ALLOWED_ORIGINS (JSON array of production frontend URL).
       - COOKIE_SECURE (must be set to true).
       - VITE_API_URL (production domain URL).
    4. Launching the App:
       - Running docker compose -f docker-compose.prod.yml up -d.
    5. Cloudflare Tunnel Configuration:
       - How to install Cloudflare Tunnel daemon (cloudflared).
       - Pointing the tunnel service to http://localhost:80 (where Nginx is listening).
    6. Backup Strategy:
       - Executing pg_dump inside the db container: docker exec -t <db_container_name> pg_dump -U securedata_user -d securedata > backup.sql.
       - Cron job syntax template for automated backups.
    7. Alternative Direct Ingress (Let's Encrypt / Certbot):
       - If not using Cloudflare Tunnel, how to expose port 80/443 on UFW.
       - Configuring Certbot for automatic certificate generation and renewal (certbot renew --quiet cron job).
