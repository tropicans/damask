<threat_model>
### Security Threat Model (ASVS L1)
- **Threat**: Brute-force attacks on the login endpoint.
  - **Mitigation**: In-memory lockout after 5 failed attempts from the same IP within 15 minutes. HTTP 423 response with clear user message.
- **Threat**: Open/unauthenticated user registration enabling unauthorized access.
  - **Mitigation**: /register endpoint requires a valid, single-use invite token. Tokens are UUID4 and expire after 48 hours.
- **Threat**: Weak passwords leading to account compromise.
  - **Mitigation**: Backend validates password policy (min 8 chars, uppercase, lowercase, digit) and returns HTTP 422 with descriptive error if policy fails.
- **Threat**: Invite token enumeration or reuse.
  - **Mitigation**: Tokens are UUID4 (128-bit entropy). Used tokens are marked is_used=True immediately upon successful registration, preventing reuse.
</threat_model>

    ```python
    import re

    def validate_password_policy(password: str) -> None:
        """
        Validates password against the application's password policy.
        Raises ValueError with a descriptive message if policy is not met.
        Policy: min 8 characters, at least 1 uppercase, 1 lowercase, 1 digit.
        """
        if len(password) < 8:
            raise ValueError("Password harus minimal 8 karakter.")
        if not re.search(r'[A-Z]', password):
            raise ValueError("Password harus mengandung minimal 1 huruf kapital, 1 huruf kecil, dan 1 angka.")
        if not re.search(r'[a-z]', password):
            raise ValueError("Password harus mengandung minimal 1 huruf kapital, 1 huruf kecil, dan 1 angka.")
        if not re.search(r'[0-9]', password):
            raise ValueError("Password harus mengandung minimal 1 huruf kapital, 1 huruf kecil, dan 1 angka.")
    ```

    2. **In-memory lockout dict and helpers** (add after validate_password_policy):
    ```python
    from datetime import datetime, timedelta
    from collections import defaultdict

    # In-memory failed login attempt tracker: { ip_address: [datetime, ...] }
    _failed_attempts: dict[str, list[datetime]] = defaultdict(list)
    _LOCKOUT_WINDOW_MINUTES = 15
    _MAX_ATTEMPTS = 5

    def _prune_attempts(ip: str) -> None:
        """Remove attempt timestamps older than the lockout window."""
        cutoff = datetime.utcnow() - timedelta(minutes=_LOCKOUT_WINDOW_MINUTES)
        _failed_attempts[ip] = [t for t in _failed_attempts[ip] if t > cutoff]

    def is_ip_locked(ip: str) -> bool:
        """Return True if the IP has exceeded the max failed attempts."""
        _prune_attempts(ip)
        return len(_failed_attempts[ip]) >= _MAX_ATTEMPTS

    def record_failed_attempt(ip: str) -> None:
        """Record a failed login attempt for an IP address."""
        _prune_attempts(ip)
        _failed_attempts[ip].append(datetime.utcnow())

    def clear_failed_attempts(ip: str) -> None:
        """Clear all failed attempts for an IP on successful login."""
        _failed_attempts[ip] = []
    ```

    ```python
    @router.post("/invite", response_model=InviteResponse, status_code=status.HTTP_201_CREATED)
    def create_invite(
        request: Request,
        current_user: User = Depends(get_current_user),
        session: Session = Depends(get_session)
    ):
        """
        Creates a one-time invite link for new user registration. Admin only.
        """
        if current_user.role != "admin":
            raise HTTPException(
                status_code=status.HTTP_403_FORBIDDEN,
                detail="Hanya admin yang dapat membuat tautan undangan."
            )
        expires_at = datetime.utcnow() + timedelta(hours=settings.INVITE_EXPIRE_HOURS)
        invite = Invite(created_by=current_user.id, expires_at=expires_at)
        session.add(invite)
        session.commit()
        session.refresh(invite)
        invite_url = f"{settings.FRONTEND_URL}/register?invite={invite.token}"
        logger.info(f"Invite created by admin {current_user.id}, expires {expires_at}")
        return InviteResponse(
            id=invite.id,
            token=invite.token,
            invite_url=invite_url,
            expires_at=invite.expires_at,
            is_used=invite.is_used,
            created_at=invite.created_at
        )
    ```
