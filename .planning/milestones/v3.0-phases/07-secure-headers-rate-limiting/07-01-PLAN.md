# Plan 07-01: Secure HTTP Response Headers Middleware

## Proposed Changes

### Backend: custom HTTP Security Headers Middleware

#### [MODIFY] [main.py](file:///c:/Users/X1%20Carbon/Downloads/Projects/self-hosted-ai-starter-kit/Dev/datamask/backend/app/main.py)
- Implement `SecurityHeadersMiddleware(BaseHTTPMiddleware)` class to inject security headers on every response:
  - `Content-Security-Policy`: `"default-src 'self'; frame-ancestors 'none';"`
  - `Strict-Transport-Security`: `"max-age=63072000; includeSubDomains; preload"`
  - `X-Frame-Options`: `"DENY"`
  - `X-Content-Type-Options`: `"nosniff"`
  - `X-XSS-Protection`: `"1; mode=block"`
  - `Referrer-Policy`: `"strict-origin-when-cross-origin"`
- Register `SecurityHeadersMiddleware` on the FastAPI application using `app.add_middleware()`.
