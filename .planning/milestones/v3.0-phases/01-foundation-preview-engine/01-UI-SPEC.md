## UI Considerations

> Populated by the ui-phase UI-consideration probe (Step 9.5) and lifted by plan-phase's
> `## UI Considerations` lift rule via the identical rule as SPEC `## Edge Coverage`. Shape-rooted UI *state*
> coverage (empty / loading / error / populated / partial / overflow / zero-one-many / long-text).

Applicable state considerations resolved: 10 covered, 3 backstop, 0 unresolved

| Category | Element(s) | Status | Resolution / Reason |
|----------|------------|--------|---------------------|
| empty | PreviewTable | ✅ covered | Tabel pratinjau disembunyikan jika tidak ada file yang dimuat (kondisi default awal). |
| loading | PreviewTable | ✅ covered | Indikator loading (skeleton/spinner) ditampilkan pada area unggah file (E1) saat file sedang diproses, dan tabel pratinjau belum muncul. |
| error | PreviewTable | ✅ covered | Jika parsing backend gagal, pesan error ditampilkan di area unggah (E1), tabel pratinjau tetap disembunyikan. |
| populated | PreviewTable | ✅ covered | Menampilkan baris header dan tepat 3 baris data pertama secara terstruktur. |
| partial | PreviewTable | ✅ covered | Sel kosong/null dalam pratinjau ditampilkan sebagai string kosong ("") atau placeholder visual abu-abu ringan agar tidak merusak tata letak. |
| overflow | PreviewTable | 🧪 backstop | Menggunakan container scroll horizontal (overflow-x-auto) jika jumlah kolom melebihi lebar layar. |
| zero-one-many | PreviewTable | ✅ covered | Hanya menampung pratinjau 3 baris data pertama (maksimal). Jika file memiliki < 3 baris, tampilkan baris yang ada (1 atau 2). |
| long-text | PreviewTable | 🧪 backstop | Teks kolom yang sangat panjang dipotong dengan ellipsis (truncate max-w-xs) untuk menjaga kerapian grid visual. |
| loading | Dropdown menu | ✅ covered | Dropdown dinonaktifkan (disabled state) saat file sedang diproses/loading. |
| error | Dropdown menu | ✅ covered | Jika rekomendasi gagal dimuat, aturan fallback "No Masking" dipilih sebagai default. |
| overflow | Dropdown menu | 🧪 backstop | Menu dropdown menggunakan scroll vertikal jika daftar opsi masking bertambah panjang. |
| long-text | Dropdown menu | ✅ covered | Label opsi menggunakan teks statis pendek (misal: "Fake Name") sehingga tidak memicu overflow teks panjang. |
| unclassified | Dropzone | ✅ covered | Tampilan dropzone menampilkan feedback visual hover (active border color change) ketika file ditarik di atasnya. |
